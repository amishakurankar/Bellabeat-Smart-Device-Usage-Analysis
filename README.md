**Capstone Case Study : Bellabeat Smart Device Usage Analysis**

**Author**

Amisha Kurankar

**Introduction**

Bellabeat is a health-focused technology company that develops smart devices to help women monitor their health and wellness. This project analyzes Fitbit Fitness Tracker data to understand users activity patterns and identify insights that can help Bellabeat improve its products and marketing strategies.

**Business Task**

Analyze smart device usage data to answer the following questions:

What are the current trends in smart device usage?

How can these trends apply to Bellabeat customers?

How can these insights help Bellabeat improve its marketing strategy?

**Dataset**

Dataset: Fitbit Fitness Tracker Data (Fitabase)

**Files used:**

hourlySteps_merged.csv

hourlyCalories_merged.csv
